\# Contribuindo



Obrigado por considerar contribuir com o Desktop Gadgets.



\## Como contribuir



\* Faça um Fork do projeto.

\* Crie uma nova branch para sua funcionalidade.

\* Faça commits claros e objetivos.

\* Envie um Pull Request descrevendo suas alterações.



\## Padrões



\* Utilize Python 3.11 ou superior.

\* Mantenha o código organizado.

\* Documente novas funcionalidades.

\* Teste antes de enviar alterações.



Toda contribuição é bem-vinda.
